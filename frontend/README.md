HOW TO RUN:

-Make sure your terminal is opened at folder 'f25-santorini-dinoflask' 
-cd to the backend folder with % cd santorini

In backend folder, run:
mvn install
mvn exec:exec

open a new terminal in folder 'f25-santorini-dinoflask'
-cd to the frontend folder with % cd frontend

In frontend folder, run:
npm install
npm start

Santorini will now start.
